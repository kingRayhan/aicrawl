## What is GraphCrawl?

GraphCrawl is a web crawling API that extracts:

- **Metadata** - Open Graph, Twitter Cards, JSON-LD, and standard meta tags
- **Markdown** - Clean markdown conversion of web pages
- **HTML** - Readability-extracted, article-focused content

Built on Cloudflare Workers for global edge computing and low latency.

## Key Features

✅ Fast metadata extraction  
✅ Clean markdown conversion  
✅ Readability-based content extraction  
✅ HATEOAS-compliant REST API  
✅ Edge computing for global performance
